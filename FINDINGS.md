# Key Findings

## ML Model Performance
| Model | MAE | R2 |
|---|---|---|
| Linear Regression | ~60 min | 0.57 |
| Random Forest | ~16 min | 0.97 |
| Gradient Boosting | ~10 min | 0.99 |
| XGBoost | ~9.4 min | 0.99 |

## Key Insights
- Queue length is the strongest predictor of wait time
- Doctor shortage nearly doubles average wait time
- Mass casualty events cause severe queue buildup
