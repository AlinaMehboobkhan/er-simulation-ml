# ER Simulation & ML Wait Time Predictor

A SimPy-based Emergency Room simulation with ML wait time prediction.

## Features
- Discrete Event Simulation using SimPy
- Time-varying patient arrivals (Poisson process)
- Priority queue for critical patients
- ML models: Linear Regression, Random Forest, Gradient Boosting, XGBoost
- Interactive Plotly dashboard
- What-if scenario analysis
